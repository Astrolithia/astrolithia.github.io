Raycast使用指北

安装方式

  1. 官网安装，直接点击官网的下载即可

  2. 通过Homebrew，直接利用命令行安装

    brew install --cask raycast

窗口管理

我用它代替了系统自带的聚焦（Spotlight）

简单介绍它自带的几个功能：

  • 快捷启动应用程序
  • 窗口管理
  • 快捷启动

下面让我来详细的介绍这几个部分

快捷启动程序

可以通过快捷键呼出输入框，搜索本机安装的应用程序

接着按下Command+K或者点击右下方的Actions可以呼出动作菜单

功能包括

  • 显示/复制路径
  • 快捷卸载软件

快捷调用AI

来到商店，搜索store

按下回车，搜索ChatGPT，如图下载量最多的那个就是

来到设置，搜索ChatGPT，填写你的API，如果有用中转站需要在API Endpoint填写

如果用的不是官方API而是中转站的记得勾选这里的Use API Endpoint并且填写API Endpoint

输入Command+K->选择Create Model

![](https://cdn.jsdelivr.net/gh/Kapemni/image@main/CleanShot 2024-10-30 at 20.36.09@2x.png)

我取了名叫翻译，提示词借用的是lobechat里的，模型用的是gpt-4o-mini（便宜），填写完毕后Command+Enter保存

输入aq，选择Ask Question

这里节选一篇考研英语，可以看到翻译后的结果

GUI For Sing-box使用教程

访问GitHub，https://github.com/GUI-for-Cores/GUI.for.SingBox/releases/tag/v1.8.9，选择对应系统的安装包

下载完后打开打开

点击Profiles->Add

新建一个名为Config的配置文件

点击Subscriptions->Add

按照图片填写（仅供参考）

点击Plugins->Plugin-Hub

先下载节点转换，直接点击节点转换下的Add，然后关闭

复制我在群里发的节点，然后点击节点转换下方的Run

然后复制下转换后的内容，选择Subscriptions，右击Sub，选择Edit Proxies(Source)，将刚刚复制后的内容粘贴上去

选择Setting-Kernel，两个都选择Update

来到OverView->Click to start

勾选右上角的System Proxy

Proxy Groups

选择Select右方的书写

选中Sub（如果后续有添加其他订阅，也需要勾选上），点击Save

照例选择Save

回到首页，点击最下方的controller

点开Select，选中新添加的节点，然后就可以正常

可以正常访问谷歌了

Mac使用rclone将alist挂载到本地

这里我犯了一个错误（包管理器安装的rclone无法将云盘挂载到本地）

输入rclone config

输入n，新建配置

输入55，选择WebDAV（这里的数字看具体情况，可能你和我不一样）

输入url，记得加端口号和dav

输入7，选择Other site/service or software

输入用户密码

输入y

输入密钥，没有的话直接按回车

是否要编辑高级配置

检查配置是否正确，确认后输入y

按q，退出配置

输入rclone lsd alist:查看是否挂载成功

输入rclone mount {配置名称}:/ {挂载地址} --copy-links --no-gzip-encoding --no-check-certificate --allow-other --allow-non-empty --umask 000 --use-mmap --daemon，将挂载到本地，这里我输入的是

要保证挂载地址在本地存在，如果不存在，请创建文件夹

可以看见alist已经挂载到本地
