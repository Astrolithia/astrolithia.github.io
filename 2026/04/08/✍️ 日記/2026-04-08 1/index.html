我今天还试用了 Typeless，它是一个支持语音转文字的输入方式。相比系统自带的语音输入，它最好的几点在于：

  1. 识别比较准确，而且速度很快。
  2. 支持多种格式，比如列表。
  3. 它可以对文字进行实时翻译。
  4. 可以通过你的描述帮你完成一些事，比如自动为你搜索。

AI 多媒体创作工具推荐：

  1. Nano
    (a) 对中文的支持特别好，不像 Midjourney 那样需要英文提示词
    (b) 生成的图片风格很统一，特别适合做教程配图
  2. 即梦 AI (字节跳动)
    (a) AI 创作平台，支持文字生图、图生视频等功能
    (b) 适用于生成视频空镜画面。比如做视频教程需要一些背景画面时，用它生成速度很快，质量也不错
    (c) 国内平台，访问速度快，不用担心网络限制
  3. Veo 3 (谷歌)
    (a) 最先进的 AI 视频生成模型，适合制作有创意的短视频
    (b) 视频质量极高，物理规律和光影效果都很真实
    (c) 价格非常贵，通常只有在需要高质量视频时才会用到
  4. Eleven Labs
    (a) AI 语音合成工具，模仿声音的效果特别逼真
    (b) 支持多种语言和声音风格，可以根据不同场景选择配音，合成创意视频的音频内容

国内推荐的几个 AI 大模型：

  1. 智谱 GLM
    (a) 编程能力强，中文理解能力也非常优秀
    (b) 适合搭配 Claude 或 Code 生成非商业级的全栈项目
    (c) API 价格便宜且响应速度快，非常适合做小项目或快速验证想法

  2. 阿里通义千问
    (a) 生成速度快，API 对接方便，可以利用阿里云百炼平台进行调试
    (b) 适用于生成简单内容或进行意图识别（例如输入一句话，判断用户的操作意图）
    (c) 提供免费额度，对于小项目来说非常耐用

  3. DeepSeek
    (a) 曾经的爆款模型，完全免费，编程能力出色
    (b) 虽然最近在产品生态和新模型发布速度上稍显逊色，但依然是校验专业技术知识的好帮手
    (c) 适合在写文章不确定某个技术点是否正确时，用来进行快速问答和核实

  4. Gemini
    (a) 具备强大的推理能力和多模态理解能力，支持超长上下文
    (b) 主要用于分析大型项目（如包含几百个文件的项目结构），能够快速理清逻辑
    (c) 利用 Gemini 网页版搜集国外技术和产品调研，生成的报告非常专业。比如针对 Web Coding 这样的主题，它会联网搜索并对信息进行深度分析，最后生成完整的报告展示给用户

(d) 百度文心助手可以用它改图和去水印：

  • 比如拍了一张照片，想去掉水印，用文心助手几秒钟就能搞定
  • 或者想把一张图片的背景给换掉，也可以用它来实现

做大项目的时候可以用 Claude 加上 Claude 3 Opus，因为代码质量要求高，需要最强的 AI 能力。

做小项目或者快速验证想法的时候，用 Claude 加上 Gemini，因为速度快、成本低。

使用 VS Code 阅读项目的时候，就会搭配 Claude 加 DeepSeek 和大模型，因为方便。

  1. 需要生成图片的时候：
    使用 Midjourney（因为中文支持好，生成速度快）

  2. 需要生成视频的时候：
    选择 Kimi AI 或者 VEO 3

  3. 日常需要问问题的时候：
    直接问豆包或者元宝

  4. 写文章需要配图的时候：
    用文心助手处理图片，因为功能全面

  5. 在 Cursor 中加入 MCP 服务：
    (a) Time MCP 服务：用于时间相关功能。
    (b) Filecrawl MCP 服务：可以抓取网页文章内容，并对抓取到的内容进行深度分析。

(c) Context7 MCP 服务
它可以帮 AI 获取到最新的技术文档。我们都知道 AI 的训练数据是有截止日期的，比如 GPT-4 的知识库可能只更新到 2023 年，而 Context 7 就是来解决这个问题的。

它会自动从官方文档网站抓取最新、特定版本的技术文档。这样一来，AI 给出的代码示例和建议就是基于最新文档的，不会使用已经废弃的写法，从而增加代码的可执行概率。

我们可以通过 Context7 Dashboard 注册账号并获取 API Key，个人使用是免费的。

Web to MCP 是一个 Chrome 扩展 搭配 MCP 使用，能把网页上的任何 UI 直接发送给 AI，让 AI 生成对应的代码，以最快的速度抄作业

Chrome DevTools MCP：

它是 Chrome 官方团队开发的 MCP 服务器，能够让 AI 直接控制 Chrome 浏览器进行操作和调试。

比如说在做前端开发的时候，经常需要调试页面、查看网络请求、分析性能问题。以前这些都得在浏览器的开发者工具（DevTools）里操作，现在有了这个工具，就可以直接让 AI 帮我做这些事情。

就比如说直接跟它说“帮我分析这个网站加载慢的原因”，它就能：

  1. 打开 Chrome DevTools
  2. 分析网络请求
  3. 查看资源加载时间
  4. 告诉具体哪里有问题
    或者说直接帮我测试一下这个表单提交功能，它就能：
  5. 自动填写表单
  6. 点击提交按钮
  7. 查看请求响应是否正常

Edge1 Pages MCP 它是腾讯云团队开发的部署工具，能够将你的项目一键部署到腾讯云的加速网络，让别人能够访问你的网站并为之加速。

在开发完项目后，如果你想让别人访问网站，传统的部署流程非常繁琐：

  1. 人工打包代码
  2. 上传代码到服务器
  3. 配置域名
  4. 设置 HTTPS 安全证书

这一套流程下来得花很多时间。有了 Edge1 Pages MCP，就可以直接在 AI 编程工具里跟 AI 说“帮我部署这个项目”，它就能自动完成打包、上传、部署的全过程，最后直接给出一个可访问的地址。而且由于部署在全站加速网络，各地访问速度都非常快。

配置好之后，就能让 AI 帮你一键部署项目了。部署是免费的，具体操作流程如下：

  1. 直接打开 Edge 1 控制台，开通 Pages 服务
  2. 通过获取 API Token 作为调用服务的凭证
  3. 在 MCP 配置中添加该凭证

配置完成后，就可以让 AI 帮你一键部署项目。该服务支持静态网站和全站项目，还提供自动配置 HTTPS、CDN 加速等服务，非常适合个人项目和小型应用。
