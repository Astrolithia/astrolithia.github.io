开发实践

  • 用 Claude Code 给万能视频下载器新增字幕时间戳跳转功能，初步测试发现点击时间戳后无法跳转到对应时间点，问题尚未解决
  • 学习了 AI 并行开发提示词技巧：在提示词末尾加上并行指令，让 AI 同时处理多个需求

工具安装 & 发现

  • 安装 Macpow：Apple Silicon 实时功耗监控 TUI 工具，可查看各组件功耗、温度、频率、利用率

    来源：https://github.com/k06a/macpow?tab=readme-ov-file

  • 安装 Chops：macOS AI 编码助手技能管理器，统一管理 Claude Code / Cursor / Windsurf / Copilot / Aider 等工具的 Skills

    来源：https://github.com/Shpigford/chops

  • 发现 V2s：https://github.com/franklioxygen/v2s

  • 发现 Git-Rewriter-Commit：AI 驱动的 Git 提交信息自动重写工具，支持 Conventional Commit 规范、Dry Run、备份分支、分批处理

    来源：https://github.com/f/git-rewrite-commits

  • 发现 Surge：在 Claude 终端中下载任意文件

    来源：https://github.com/surge-downloader/Surge

其他

  • 加入 Claude Code Team 车（3 元），节省使用成本

🧠 学到了什么（知识/成长）

AI 编程技巧

  • 提示词引导解析库选择：可引导 AI 使用 ByteMD 等 Markdown 解析库实现特定功能
  • 并行开发提示词：在需求末尾加上”如果同时给了你多个需求，你需要合理地规划所有的需求任务并行开发，用更快的速度完成任务，并且要注意代码之间不要冲突，不要影响已有功能的正常使用”
  • 跨 AI 定位问题：自己不确定问题出在哪时，可以问其他 AI 帮忙定位

Claude Code Token 优化

  • 使用 /context 命令监控 Token 消耗分布
  • 6 条节省 Token 的方法：
    1. 关闭不需要的 MCP Server（工具描述文本占用大量 Token）
    2. 经常 /clear 清除对话历史
    3. 使用 -compact 压缩历史（长上下文场景）
    4. 经常用 /context 监控 Token 使用
    5. 精简 CLAUDE.md 文档（每次请求都会发送）
    6. 保持代码文件精简（拆分大文件、删除无用代码）

💭 思考 & 情绪

  • 时间戳跳转功能没跑通，有点挫败，需要继续排查
  • 今天工具发现比较多，但实际动手开发时间偏少

⚠️ 问题 & 阻碍

  • Claude Code Token 消耗大，需要优化使用方式
  • 今日以工具探索为主，实际编码产出较少
