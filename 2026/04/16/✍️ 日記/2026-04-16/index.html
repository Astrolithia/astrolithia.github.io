    • 对 Java 25 原生支持
    • HTTP Service Clients 出现
  • 主题 2：什么是RestTemplate 或 WebClient？HTTP Service Clients的出现带来了什么？
    • 它们是Spring 的同步 HTTP 客户端，用于调用第三方接口（别人的 HTTP API）
    • HTTP Service Clients 让你不用每次都手写”URL” “method” “ headers” “body” “response parse”
  • 主题 3：原生API 版本管理
  • 主题 4：JSpecify 空安全体系是什么？
    • 默认非空+明确标注可空（让空指针从运行时炸弹到编译器报错）
    • 在类或包上加”@NullMarked”注解，里面默认所有引用类型都是非空的，只有明确标了”Nullable”的才可能为 null
